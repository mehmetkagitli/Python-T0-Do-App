# 📝 Simple To-Do List Application (Python)

This is a simple console-based To-Do List application written in Python.

## ⚙️ Features
- View all tasks
- Add a new task
- Remove a task
- Mark a task as completed
- Data is stored in a `.txt` file

💡 Example
To-Do List Menu
1. View To-Do List
2. Add Task
3. Remove Task
4. Mark Task Completed
5. Exit
